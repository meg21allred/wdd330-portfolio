    ex. - user.sayHi = function() { alert("Hello!"); };

      • user = { sayHi: function() { alert("Hell0!"); };

    ex. - Access the user name sayHi() { alert(this.name); };

Chapter 6 The Document Object Model

  • The DOM allows you to access elements of a web page and enable interaction with the page by adding and removing elements,
    changing the order, content and attributes of elements, and even altering how they are styled.
  • getElementsById("id") - be sure that each element has a unique id or funny things will happen to your document.
  • getElementsByTagName() - will return a live node list of ALL the elements with the tag name that is provided as an argument.
  • getElementsByClassName() - will return a live node list of all elements that have the class name that is supplied as an arguement.
  • document.querySelector() - allows you to use CSS notation to find the first element in the document that matches a CSS selector
    provided as an argument.
  • set attribute and get attribute - wonderWomen.setAttribute('id', 'amazon');, wonderWomen.getAttribute('id'); = 'amazon'
      for attributes 'class' and 'for' you must used 'className' and 'htmlFor'
  • Creating an element const flash = document.creatElement('li');
    • To fill the element, create a text node - const flashText = document.createTextNode('Flash');
    • To link the element and the text node together, use appendChild() - flash.appendChild(flashText);
    • easier way - const flash = document.createElement('li'); flash.textConten = 'Flash';
    • Best to use a function so code does not become repetative -
      Function createElement(tag, text) { const el = document.createElement(tag); el.testDContent = text; return el }
    • Adding new element to the page - heroes.appendChild(flash)
    • other useful functions include insertAfter(), removeChild(), and replaceChild()
