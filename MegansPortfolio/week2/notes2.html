Chapter 3 Arrays, Logic and Loops

  • Destructuring Arrays
    • taking values out of an array and presenting them as individual values
    • allows the ability to assign multiple values at the same time.
      ex. const [x,y] = [1,2];
    • Allows the ability to swap the value of two variables without having to use a temp. variable.

      • ex. [x,y] = [y,x];
  • Spread operator ( ... ) - placed at the front of the array to spread the elements of the array and can been
    used to put two arrays together in a new array.
  • Splice() - is very useful for inserting, replacing and completely removing items from an array.
    • insert - array.splice[4(index to insert), 0 (number of items to remove before inserting), "item" (item to insert in this case a string)
    • replace - array.splice(4,1,"item")
    • remove - array.splic(4,1 (last parameter is left blank because you don't want to replace it with anything))
  • includes - array.includes("item") returns true or false and can include extra paramenter to indicare what indext to star the search from.
  • Convert Sets to Arrays - const shoppingArray = [...shoppingSet] or const shoppingArray = Array.from(shoppingSet);
  • Convert Maps to Nested arrays in much the same way as sets to arrays.
  • Ternary Operator - condition ? (//code to run if condition is true) : (//code to run if condition is false)

Chapter 4 Functions

  • Defining functions:
    • Function Declarations - Function hello() { code...}
    • Function Expressions - const goodbye = function() { code... }
      const goodbye = function bye() {code...}
    • Function Constructors - const hi = new Function('code..'); (not recommended)
    • Arrow Functions - const square = x => x*x; (always annon. so must be assigned to a variable.
  • Default parameters - assigns a default value to the parameters if no arguement is given
    ex. function hello(name='World')
    • default values should always come after not default values or the function will not work correctly.
  • Callbacks - A function that is passed as an argument to another
  • ForEach() - similiar to a for loop - ex. colors.forEach(color, index) => console.log('color at position ${index} is ${color}');
  • Map() - iterates over an array and takes a callback function as a parameter that is invoked on each item.
    ex. [1,2,3,].map(x => 2 * x);
  • filter() - returns a new array that contains only items that are true when passed to the callback.
  • Chaining iterators - using more than one iterator function to create powerful transformations of data in arrays.
    ex. const sales = [100, 200, 55];
    totalAfterSalesTax = sales.map( (amount) => amount * 1.15).reduce( (acc,val) => acc + val);
